!Bienvenido! Esta página web es un control de stock interno, beta 1.0.

Cada usuario que se logue tiene para su propio negocio de su inventario, sus clientes, sus ventas, sus movimientos y sus estadísticas.

Si no quiere loguearse, puede usar estas pestañas y encontrar como actua cada función.

Login y registro de usuarios con verificación de email, se mantiene la sesión abierta através de las cookies.

Carga de miles de productos, simple, sencillo y con muchísimos precios, alguno para minoristas, mayoristas, particulares, etc.

Carga de clientes y guardado en la nube de sus datos para siempre tenerlos.

Las diferentes estadísticas de tu local, cuantas ventas y que plata se genero por día, semana, mes y hora. Mostrados en un gráfico de barras.

En un futuro, tener una lista actualizada de cada proovedor, con sus datos de contactos y los productos en venta.

Suscripciones pagas para diferentes niveles de usuarios y funcionalidades.




En la siguiente versión se pueden seguir agregando más funcionalidades como: - Porcentaje de ganancia (Diferencia entre precio de compra y venta) - Módulo de facturación automatica (Facturas A, B, C y envio por emails) - Avisos de sotck bajo o agotado con alertas por email. - Backup en la nube automatico cada cierto tiempo - Integración con otros sistemas (Ej: MercadoLibre, Andreani para enviós, etc.) - Reportes por usuarios / vendedores, cantidad de ventas, problemas con clientes, etc.
